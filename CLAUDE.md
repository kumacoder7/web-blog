# Tool Blog Harness

이 프로젝트는 Google AdSense 수익 기반의 도구 블로그 운영을 위한 하네스입니다.
키워드 조사부터 HTML 도구 페이지 개발, QA 검수, 배포 준비까지 반복 가능하게 구성되어 있습니다.

## 주요 위치

| 목적 | 경로 |
|---|---|
| 팀원 역할 카드 | `.claude/agents/` |
| 전체 실행 스킬 | `.claude/skills/tool-blog-orchestrator/SKILL.md` |
| 산출물 | `artifacts/` |
| 산출물 지도 | `artifacts/README.md` |

## 자연어 라우팅

아래와 같은 요청이 들어오면 `tool-blog-orchestrator`를 먼저 사용한다.

- "{도구명} 계산기 페이지 만들어줘"
- "{도구명} 변환기 블로그 페이지 새로 만들어줘"
- "이전에 만든 {도구명} 도구에 기능 추가해줘"
- "{도구명} 페이지 QA만 다시 해줘"
- "키워드 조사부터 다시 해줘"
- "도구 HTML 수정해서 재검수해줘"
- "다음 도구 페이지 기획해줘"

단순 질문, 일반 코드 수정, 하네스 구조 설명 요청에는 orchestrator를 사용하지 않는다.

## 플랫폼 및 기술 스택

- **호스팅**: Cloudflare Pages (무료, 무제한 대역폭)
- **파일 형태**: 단일 HTML (CSS/JS 인라인 포함)
- **광고**: Google AdSense (배포 전 publisher-id 입력 필요)
- **빌드 도구**: 없음 (정적 HTML 직접 배포)

## 사람 승인 지점

1. **기획서 승인** (Phase 3): 기능 범위와 계산 공식 확정
2. **배포 승인** (Phase 6): QA 통과 후 최종 HTML 확인
